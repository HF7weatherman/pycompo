# pycompo
A python-based toolbox for composite analysis.

### Credits
The basic idea and working principle of the composite analysis is adapted from
[X. Chen et al. (2023)](https://journals.ametsoc.org/view/journals/atsc/80/12/JAS-D-23-0075.1.xml).

Functions and files in `pycompo/core/cluster.py` are from Sebastian Ortega`s library *pyorg*.
